개발 흐름을 끝까지 이어가는 AI 코딩 에이전트

SuperLiora는 계획, 조사, 목표 관리, 병렬 실행, 검증, 기억을 하나의 작업 흐름으로 연결합니다. 긴 세션에서도 맥락을 보존하고, 근거 있는 결정을 남기며, 개발자가 다음 단계에 집중할 수 있도록 돕습니다.

Problem

긴 작업은 맥락을 잃고, 결정은 흩어집니다

복잡한 구현, 마이그레이션, 리팩터링은 중간에 목표가 흐려지고, 근거가 사라지며, 같은 질문을 반복하게 됩니다. 도구가 많아질수록 오히려 흐름이 끊깁니다.

Migration & refactoring

대규모 변경에서 계획, 영향 범위, 검증을 하나의 흐름으로 유지합니다.

Feature implementation

요구사항을 명확히 고정하고, 구현 중에도 완료 기준을 잃지 않습니다.

Incident & debugging

조사 근거와 시도한 방법을 정리하며 루트 원인에 집중합니다.

Documentation & knowledge

검증된 결정을 다음 작업에서 다시 사용할 수 있는 자산으로 남깁니다.

Solution

Shift-Tab 한 번으로 Ultrawork 시작

일반 프롬프트는 가볍게, 복잡한 작업은 Ultrawork로 전환합니다. UltraPlan interview부터 UltraGoal, research, Swarm decision, verification, learning까지 이어집니다.

Ultra workflow

계획부터 검증까지 이어지는 실행 구조

Shift-Tab Ultrawork mode는 복잡한 요청을 UltraPlan interview로 먼저 좁히고, true/false로 검증 가능한 UltraGoal을 만든 뒤 research, Swarm decision, integration, verification 순서로 진행합니다.

UltraPlan 01

완료 여부가 명백해질 때까지 요구사항, 제약, 위험을 인터뷰합니다.

UltraGoal 02

계획이 승인된 뒤 구체적 목표와 검증 기준을 고정합니다.

Swarm decision 03

Memory and LLM Wiki

작업의 맥락을 기록하고 다시 활용합니다

Liora Recall은 프로젝트 사실, 결정, 절차, 후속 작업을 기억합니다. LLM Wiki는 코드베이스와 검증 근거를 문서화해 다음 작업이 같은 출발점을 공유하도록 돕습니다.

Semantic memory facts

프로젝트 구조, API 제약, 팀 규칙을 보존합니다.

Procedural memory how-to

반복되는 빌드, 배포, 검증 절차를 다시 사용할 수 있게 정리합니다.

Governance memory rules

중요한 정책과 금지 사항을 이후 작업에서도 확인할 수 있게 유지합니다.

Premium operator surface

오래 머무는 터미널을 더 읽기 쉽게

긴 세션에서는 시각 피로, 코드 가독성, 상태 인지가 작업 품질에 영향을 줍니다. SuperLiora는 preset, 외부 terminal palette, syntax-aware color를 정제된 TUI 경험으로 제공합니다.

Neon Operator

Premium capabilities

현장에서 필요한 기능을 한 흐름에 담았습니다

SuperLiora는 계정과 API, context, research, memory, theme, workflow를 따로 관리하지 않고 같은 에이전트 흐름에서 다룹니다.

01

Provider routing

여러 API key와 OAuth account를 등록하고 quota, cooldown, latency 상태를 기준으로 fallback 후보를 선택합니다.

02

Local web research

외부 검색 경로가 충분하지 않을 때 local fetch/search fallback으로 필요한 근거를 보완합니다.

03

Context OS

긴 세션을 structured working memory, repair, bounded rehydration으로 관리합니다.

04

LLM Wiki

코드베이스 지식, 조사 근거, 검증 결과를 다음 작업에서 다시 쓸 수 있는 문서 자산으로 정리합니다.

05

UltraGoal

목표, 예산, 완료 조건을 명확히 두어 장기 작업의 방향을 유지합니다.

06

ACP ready

터미널과 호환 editor에서 같은 SuperLiora workflow를 이어갈 수 있습니다.

Install

소스에서 설치하고 liora로 시작하세요

설치 스크립트는 SuperLiora를 독립 명령으로 빌드하고 실행 경로를 설정합니다. macOS/Linux는 curl, Windows는 PowerShell one-liner로 시작할 수 있습니다.

$ curl -fsSL https://raw.githubusercontent.com/claudianus/superliora/main/install.sh | bash
PS> irm https://raw.githubusercontent.com/claudianus/superliora/main/install.ps1 | iex
$ liora

Shift-Tab로 Ultrawork를 시작해 보세요
