AI coding harness

A terminal-first AI coding harness for long, complex tasks

SuperLiora runs in your terminal. It connects planning, research, goal control, parallel execution, verification, memory, documentation, browser use, and computer use in one continuous workflow. You stay in the same context from the first question to the last verification.

Problem

Long tasks lose context and scatter decisions

Complex implementations, migrations, and refactors blur the goal, lose evidence, and repeat the same questions. More tools often mean more broken flow.

Solution

Shift-Tab into Ultrawork

Use lightweight prompts for quick work. Switch to Ultrawork for complex tasks and move through UltraPlan interview, UltraGoal, research, Swarm decision, verification, and learning.

Harness capabilities

Beyond the editor: browser, desktop, and routing

SuperLiora does not stop at the editor. It drives CloakBrowser + Playwright for the web, CUA/MCP for native desktop, and routes providers by quota, cooldown, latency, and health — all inside the same harness.

Browser-use CloakBrowser + Playwright

Observe, act, screenshot, and evaluate web pages as part of the same workflow.

Computer-use CUA / MCP

Capture and act on native desktop windows for GUI debugging and automation.

Provider routing quota, cooldown, latency

Register API keys and OAuth accounts, then select fallback candidates by health.

Install

Install from source and start with liora

The installer builds SuperLiora from source and sets up the independent liora command. Use curl on macOS/Linux or the PowerShell one-liner on Windows.

$ curl -fsSL https://raw.githubusercontent.com/claudianus/superliora/main/install.sh | bash
PS> irm https://raw.githubusercontent.com/claudianus/superliora/main/install.ps1 | iex
$ liora
