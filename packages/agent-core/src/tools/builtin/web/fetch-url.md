Fetch content from a URL. For HTML, main article text is extracted; for plain-text/markdown, the full body is returned verbatim. The result states which form you got. Only fully-formed public `http`/`https` URLs are supported; other schemes and private/loopback addresses are not fetched. Very large pages may be truncated or refused. If extracted content is incomplete (needs rendering or targeted selection), use an available Scrapling-compatible MCP/CLI/browser tool for authorized public-page observation.
