Search the web for up-to-date information.

Include the current year from the latest `<current_time>` reminder or `GetCurrentTime` for recent releases, CVEs, news, or other time-sensitive facts.

Each result has a title, publication date when available, URL, and snippet. With `include_content`, full page text may follow the snippet when available.

The host may use a no-subscription local search provider when managed search is not configured. For claims that affect implementation, follow promising results with FetchURL and cite primary source URLs — do not rely only on snippets.

When you rely on a result, cite its source URL so the user can verify it.
