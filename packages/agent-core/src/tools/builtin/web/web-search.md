Search the web for information. Use this when you need up-to-date information from the internet.

Include the current year from the latest `<current_time>` reminder or `GetCurrentTime` when searching for recent releases, CVEs, news, or other time-sensitive facts.

Each result includes its title, a publication date when available, its URL, and a snippet. When `include_content` is enabled, the full page content—when available—is appended after the snippet.

The host may satisfy this tool through a no-subscription local search provider when a managed search service is not configured. For claims that affect implementation, follow promising results with FetchURL and cite primary source URLs instead of relying only on snippets.

When you rely on a result in your answer, cite its source URL so the user can verify it.
