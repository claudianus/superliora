Return the authoritative current date and time from the host clock.

Use this when you need to confirm the current year, date, or time before time-sensitive research (WebSearch, FetchURL), scheduling, or any answer that depends on "today" or "now". Do not guess the date from pretrained knowledge — call this tool or rely on the latest `<current_time>` system reminder instead.

Returns ISO time with local offset, a human-readable local timestamp, the IANA timezone name, and the UTC offset.
